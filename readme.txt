Show your Picturelife albums direct on your web site without any HTML coding.

== Description ==

This plugin has been designed to help anyone show their Picturelife photo albums direct on their own WordPress site without the need to install any code.

Simply download and install the plugin and when ready generate a shortcode via the backend.

== Installation ==

1. Upload the folder `shortcodes-for-picturelife` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Generate a shortcode via the admin menu Picturelife in WordPress.
4. Paste the shortcode into the post or page you'd like to use it on.

== Frequently Asked Questions ==

= Do i need to have an account to use this plugin? =

Yes you must be registered with https://picturelife.com to use our plugin.
